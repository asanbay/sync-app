# TARS - Говорящий Робот

Веб-альтернатива робота TARS из фильма "Интерстеллар" с похожим стилем разговора.

## Описание

TARS (Тактический Автоматизированный Робот-Собеседник) - это интерактивный чат-бот, вдохновлённый персонажем из фильма "Интерстеллар". Робот обладает характерной для оригинального TARS личностью: остроумием, честностью и способностью к саркастичным комментариям.

## Особенности

### Настраиваемые параметры личности
- **Честность (0-100%)**: Определяет, насколько откровенным будет TARS в своих ответах
- **Юмор (0-100%)**: Контролирует уровень сарказма и шуток в ответах

### Стиль общения
TARS общается в характерном стиле:
- Прямолинейные и честные ответы при высоком уровне честности
- Остроумные и саркастичные комментарии при высоком уровне юмора
- Самоиронизация и meta-юмор
- Логичные и рациональные рассуждения
- Отсылки к научной фантастике и космосу

### Функциональность
- Ответы на различные вопросы с учётом настроек личности
- Рассказывает шутки (качество зависит от настройки юмора)
- Философские рассуждения
- Обсуждение науки, космоса и человечества
- Быстрые подсказки для начала диалога

## Использование

1. Откройте файл `tars.html` в браузере
2. Настройте уровни честности и юмора по своему вкусу
3. Введите сообщение в текстовое поле или выберите одну из подсказок
4. Наслаждайтесь общением с TARS!

## Примеры вопросов

- "Расскажи о себе"
- "Какова твоя миссия?"
- "Расскажи шутку"
- "Что думаешь о человечестве?"
- "Расскажи о космосе"
- "Какой смысл жизни?"

Робот TARS из фильма "Интерстеллар" (2014) режиссёра Кристофера Нолана - это один из самых запоминающихся AI-персонажей в кино. Его уникальная личность с настраиваемыми параметрами честности и юмора делает его особенным среди других роботов в научной фантастике.
